# Employee Management API (ASSIGNMENT5)

A simple RESTful API built with **Express.js** and **MongoDB** for managing employee data.  
Supports full CRUD operations.

---

## 📁 Project Structure

ASSIGNMENT5/
├── app.js
├── models/
│ └── employee.js
├── routes/
│ ├── create.js
│ ├── read.js
│ ├── update.js
│ └── delete.js
├── package.json
├── .gitignore
└── README.md

---

## 🚀 Features

- Create an employee
- Read all employees
- Update employee by ID
- Delete employee by ID

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman (for testing)
